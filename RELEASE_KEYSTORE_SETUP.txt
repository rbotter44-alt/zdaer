Release signing is now enabled in android/app/build.gradle.kts.

Keep your real files local only:
- android/key.properties
- android/app/lighton-release.jks

Example android/key.properties:
storePassword=YOUR_PASSWORD
keyPassword=YOUR_PASSWORD
keyAlias=lighton
storeFile=lighton-release.jks

Build:
flutter clean
flutter pub get
flutter build apk --release --obfuscate --split-debug-info=build/symbols --tree-shake-icons --dart-define=APP_STRICT_SECURITY=true

If an older debug-signed version is installed, uninstall it once:
adb uninstall com.lighton.app
