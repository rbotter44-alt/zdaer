# نسخة PWA / Web

هذه النسخة أضيفت لها ملفات الويب المطلوبة حتى تعمل كـ Flutter Web PWA:

- `pubspec.yaml` على مستوى جذر المشروع.
- مجلد `web/` مع `index.html` و `manifest.json` وأيقونات PWA.
- طبقة توافق داخل `lib/pwa/` حتى لا يكسر `dart:io` بناء الويب.
- جعل تحميلات الويب تفتح الرابط في المتصفح بدل محاولة الكتابة على ملفات الجهاز، لأن PWA لا تملك نظام ملفات أندرويد.

## التشغيل

```bash
flutter pub get
flutter run -d chrome
```

## البناء

```bash
flutter build web --release
```

الناتج سيكون داخل:

```text
build/web
```

## ملاحظات مهمة

- نسخة أندرويد بقيت كما هي قدر الإمكان، والطبقات الجديدة تعمل فقط لتسهيل بناء الويب.
- بعض ميزات الأندرويد مثل `MethodChannel`, PiP, SecurityGuard, تخزين الملفات المحلي، وصلاحيات التحميل لا تعمل بنفس الطريقة في PWA.
- صفحات WebView على الويب تعتمد على دعم `flutter_inappwebview`/iframe وقد تتأثر بسياسات CORS و X-Frame-Options الخاصة بالمواقع.
- ملفات الفيديو والترجمات الخارجية قد تحتاج HTTPS و CORS صحيح حتى تعمل داخل المتصفح.
